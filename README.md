## Desenvolvimento

O site é feito em [Vue](https://vuejs.org/), para usá-lo localmente, você precisará ter `npm` instalado na máquina.

### Instalar dependências

```
npm install
```

### Para iniciar o servidor

```
npm run serve
```

### Rodar os testes

```
npm run test
```

### Lint
```
npm run lint
```

### Build do projeto

```
npm run build
```
